# Preflop Practice

ポーカーのプリフロップ表を暗記するための練習アプリ。

## 技術スタック

- React 18
- Vite
- CSS (vanilla)

## プロジェクト構成

```
src/
├── main.jsx              # エントリポイント
├── App.jsx               # メインコンポーネント（状態管理、キーボード操作）
├── App.css               # グローバルスタイル（フェルト背景、ボタン、トグル）
├── components/
│   ├── Card.jsx          # カード表示コンポーネント
│   └── Card.css          # カードスタイル（フリップアニメーション）
└── utils/
    └── cards.js          # カードデッキ生成、シャッフル、ポジション定義
```

## 主要機能

- **カード表示**: 52枚デッキからランダムに2枚を抽出
- **ポジション**: UTG, HJ, CO, BTN, SB, BB からランダム選択
- **操作**: 「次のハンド」ボタン または スペースキー
- **トグル**: ポジション表示のON/OFF切り替え

## 開発コマンド

```bash
npm run dev      # 開発サーバー起動
npm run build    # 本番ビルド
npm run preview  # ビルドプレビュー
```

## デザイン

- カジノ風テーマ（ダークグリーンのフェルト調）
- アクセントカラー: ゴールド (#d4af37)
- レスポンシブ対応（モバイル/デスクトップ）
